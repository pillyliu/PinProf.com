PinProf 3.6.0

The latest PinProf update adds the entire Open Pinball Database to the Library, improves game and asset matching, and upgrades GameRoom and Settings.

Latest update

Version 3.6.0

Current
  • Added PinTips from Match Play
  • Browse the entire OPDB catalog in Library
  • Improved Library loading and scrolling performance
  • Improved matching of games with their rulesheets and image assets
  • Added editable Ownership details in GameRoom, including purchase date and purchase price
  • Added a Settings toggle to show or hide the Lansing Pinball League tab
  • Added a Help improve PinProf section in Settings with GitHub issue and email feedback options
